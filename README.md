# **Backend Server for**[ **Furnitura App**](https://github.com/khayrullaev/furnitura-front "Furnitura App")


## Tech-stack

* NodeJS / Express
* MongoDB / Mongoose
* JWT Authentication
* Nodemailer
* JOI schema validation


## Features

* Email Sign Up
* Email Confirmation
* Email Login
* Forget Password
* Reset Password
* Store / Products / Profile queries
* Create Orders
