# **Backend Server for**[ **Furnitura App**](https://github.com/khayrullaev/furnitura-front "Furnitura App")


## Tech-stack

* NodeJS / Express
* MongoDB / Mongoose
* JWT Authentication
* Nodemailer
* JOI schema validation


## Features

* Sign Up
* Email Confirmation
* Login
* Forget Password - with OTP
* Reset Password
* Store / Products / Profile queries
* Create Orders
